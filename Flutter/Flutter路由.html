# Flutter路由

路由即页面跳转，本篇主要介绍两个方面：路由导航__和__路由传值

首先Flutter的路由导航分以下三种：

# 动态路由

动态路由无需在MaterialApp内的routes中注册即可直接使用，示例如下：

    //从当前页跳转到HomePage
    Navigator.of(context).push(MaterialPageRoute(
        builder: (context) => Homepage(),
    ));

    //返回
    Navigator.of(context).pop();

# 静态路由

需要先在MaterialApp中申明路由名称以及对应跳转的页面，示例如下：

    return MaterialApp(
      routes: {
        "/login":(context)=>LoginPage(),
        "/home":(context)=>HomePage(),
        "/my":(context)=>MyPage(),
      },
      home:RootPage()
    );

# 路由拦截

    onGenerateRoute: (RouteSettings settings) {
        String routeName = settings.name;
        switch (routeName) {
            case "/":
                return MaterialPageRoute(builder: (context) {
                    return HomePage(settings.arguments);
                });
            case "settings":
                return MaterialPageRoute(builder: (context) {
                    return LoginPage(settings.arguments);
                });
            default:
                return MaterialPageRoute(builder: (context) {
                    return RootPage(settings.arguments);
                });
    }

路由传值包括页面跳转传值和页面返回传值

页面跳转传值：

  1. 构造函数传值
    //LoginPage -> Homepage
    Navigator.of(context).push(MaterialPageRoute(
        builder: (context) => Homepage(title:'首页'),
    ));
    //HomePage 取值
    class HomePage extends StatelessWidget {
        HomePage({Key key, this.title}) : super(key: key);
        final title;
    }
}
  1. ModalRoute传值
    //LoginPage -> Homepage
    Navigator.of(context).push(MaterialPageRoute(
        settings: RouteSettings(name:"home",arguments:"首页"}),
        builder: (context) => HomePage(),
    ));
    //HomePage 取值
    dynamic arguments = ModalRoute.of(context).settings.arguments;

页面返回传值

调用pop时直接传参:

    // Homepage-> LoginPage
    Navigator.of(context).pop("Test");

    //LoginPage 取值
    Navigator.of(context).push(MaterialPageRoute(
        builder: (context) => LoginPage(),
    )).then((value){ // 获取pop的传值
        print(value);
    });
