# iOS APP安装包瘦身实践

# 删除无用的资源

  1. LSUnusedResources：删除一些弃用业务的图片。

  2. fdupes：可以在指定的目录及子目录中查找重复的文件。fdupes通过对比文件的MD5签名，以及逐字节比较文件来识别重复内容。

  3. tinypng：无损压缩图片，通过有选择地减少图像中的颜色数，存储数据所需的字节数更少。

# 可执行文件瘦身

LinkMap文件是Xcode产生可执行文件的同时生成的链接信息，用来描述可执行文件的构造成分，包括代码段（__TEXT）和数据段（__DATA）的分布情况。只要设置Project->Build Settings->Write Link Map File为YES，build完后就可以在设置的路径看到LinkMap文件了。

我们可以用脚本从linkmap中统计出每个.o目标文件占用的体积和每个.a静态库占用的体积

主要是删除静态库的armv7s、x86架构，只保留arm7和arm64.
